### Commands ###

- /guilds info [guild]/[tag]
- /guilds list

X /guild create [guild] [tag] (leader)
X /guild invite [player]
- /guild kick [player]
- /guild edit [option] [value]
  - name | string
  - tag | string (<= 4 chars)
- /guild leave
X /guild disband (guild)

X /guild accept
X /guild deny

- /guild reload

X /grank create [rank]
X /grank delete [rank]
- /grank edit [rank] [option] [value]
  - colour | enum
  - name | string
  - invite | boolean
  - kick | boolean
  - adminchat | boolean
  - tp | boolean
  - tphere | boolean
  - default | boolean

- /grank set [player] [rank]
- /grank info [rank]
X /grank list

X /g [msg] - Guild chat
X /ga [msg] - 'guild admin' chat for 'admins' in guilds

- /hq set
- /hq

- /gtp - Teleport to guild member
- /gtphere - Teleport guildmember here

X Guild Tags in chat

### Bukkit Perms ###
noirguilds.create # Ability to create guild (under own name)
noirguilds.create.other # Can create guild under another's name
noirguilds.disband.other # Can disband another guild
noirguilds.reload # Reload plugin and config files
